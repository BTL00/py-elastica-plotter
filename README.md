# py-elastica-plotter

Open-source Python deflection curve plotter based on    Leonhard Euler   and   Jakob Bernoulli  theory for elastic lines.



## Requirements:
- Python 2.7
- PIL
- NumPy 
- Tkinter
- matplotlib
- SciPy

## Run:

```python python-2.py``` / ```python2.7 python-2.py```
